image.pixel_clusters¶

datapipe.image.pixel_clusters.get_islands(array, threshold=0.2)[source]¶

…

Parameters:
  • array (Numpy array) – The input image to clean.
  • threshold (float) – The “level of the sea” before island cleaning.
Returns:

  • Numpy array – filtered_array the input image with all pixels below threshold put to 0 (may contain NaN values).
  • Numpy array – label_array define the island id each pixel belongs to (doesn’t contain NaN values).
  • Integer – num_labels the number of islands.

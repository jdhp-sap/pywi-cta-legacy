denoising.tailcut¶

Denoise FITS and PNG images with the tailcut algorithm.

Example usages:
./denoising_with_tailcut.py -h ./denoising_with_tailcut.py -T 0.5 -t 0.1 ./test.fits ./denoising_with_tailcut.py -T 0.5 -t 0.1 ./test.fits ipython3 – ./denoising_with_tailcut.py -t 0.0001 -s ./test.fits

This snippet requires Numpy, Matplotlib and PIL/Pillow Python libraries.
